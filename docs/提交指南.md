提交指南



本文档为希望提交其数字产品以在Marketplace上发布的制作者提供了全面的指南。通过遵守这些准则，您可以确保您的提交对Ulanzi Studio 生态系统做出积极贡献，在尊重法律和社区标准的同时为用户提供价值。
安全、安保和隐私
您的产品不得危及用户安全或设备完整性。这包括避免使用可能导致过热、未经授权的关机或引入有害软件的软件。
您的产品必须采用强大的安全措施来保护用户数据并防止未经授权的访问。如果您的产品使用用户提供的数据，请确保使用行业标准加密、安全身份验证方法和定期安全更新。
如果适用，请提供清晰、可访问的隐私权政策，概述如何收集、使用和保护用户数据。产品必须遵守最高的数据隐私标准，并确保遵守 GDPR、CCPA 和其他相关数据保护法规。
如果您不确定是否在您的产品中包含此信息，请随时在 Ulanzi Discord 中询问我们。


产品独特性和功能性
Ulanzi Studio Marketplace 庆祝创新。与现有产品过于相似、没有独特功能或没有明显用户优势的产品可能不被接受。


内容适当性
我们致力于打造一个包容性的 Marketplace。不接受包含不当、冒犯性或歧视性内容的商品。所有提交的内容都应尊重我们多样化的用户群。


法律合规性和Ulanzi Studio的政策
您的产品必须符合所有法律要求和 Ulanzi Studio的政策。确保您有权使用或分发产品中的所有内容，并遵守 Ulanzi Studio品牌指南。


提交前的准备
彻底测试您的产品是否存在错误、错误或崩溃。确保您的商品描述、媒介和联系方式准确完整。


社区准则合规性
遵循Ulanzi Studio的社区准则，避免垃圾邮件、尊重知识产权并确保您的产品不含令人反感的内容。


产品可用性
商品应该是有用的、有价值的，而不是为了让用户感到不适而设计的。避免使用临时内容，确保可访问性，并防止对软件或硬件交互产生负面影响。


知识产权注意事项
作为我们尊重他人知识产权承诺的一部分，Ulanzi Studio Marketplace 严禁在未经明确授权的情况下发布侵犯受保护第三方材料的数字产品。每位手艺人都有责任确保其商品符合所有适用的知识产权法律和指南。
如果产品受到知识产权投诉或被认定为可能侵权，Ulanzi Studio保留从 Marketplace 中删除涉案内容的权利，以等待彻底调查。此行动是我们遵守（DMCA） 的一部分，该法案旨在保护版权和知识产权所有者的权利。

知识产权侵权行为的举报和解决
为了维护 Ulanzi Studio Marketplace 的完整性并确保为我们的社区提供安全、富有创意的环境，我们鼓励举报任何违反我们准则的行为。如果您认为商品侵犯了您的知识产权或违反了我们的社区标准，我们要求您遵循我们的社区准则中概述的举报程序。
当社区成员之间就知识产权侵权发生争议时，我们建议直接与相关个人或组织解决。如果这些努力没有得到令人满意的解决方案，我们鼓励受影响的各方按照我们既定的程序提交 DMCA 报告。
Ulanzi Studio致力于与执法机构合作，并在我们认为存在人身伤害风险或对公共安全构成威胁时采取必要行动。


变现规则
禁止外部付费墙和支付机制：在 Ulanzi Studio Marketplace 上发布的产品不得实施自己的付费专区，也不得将用户引导至 Ulanzi Studio Marketplace 之外的支付机制。这确保了交易在整个平台上保持安全、透明和一致，为我们的用户提供值得信赖和熟悉的体验。
广告限制：为了保持 Ulanzi Studio生态系统中用户体验的完整性，严禁在产品内展示广告。此策略旨在防止用户体验中断，并保持 Marketplace 和任何 Ulanzi Studio应用程序的美学和功能质量。
异常：考虑到数字产品供应的多样性和我们手艺人的不同商业模式，我们可能会根据具体情况考虑上述规则的例外情况。这些例外情况需要经过 Ulanzi Studio的审查和批准，以确保它们符合我们的社区标准和总体 Marketplace 目标。
如果制作者有兴趣探索可能属于这些例外情况的创收模式，我们鼓励他们直接联系Ulanzi Studio进行咨询和批准。此流程可确保所有商品都符合我们的指南，同时允许采用创新和灵活的变现策略，使制作者和用户都受益。


产品更新
使用与初始提交相同的工具提交产品的新版本。包括版本控制详细信息和更新的文件。


拼写和语法
确保您的商品描述没有拼写错误和语法错误。利用校对工具保持专业精神。


将灵感归功于
如果他人的作品激发了您的商品的灵感，请提供适当的信用信息并遵循任何署名要求。


用户体验和设计
高质量的设计和无缝的用户体验至关重要。您的产品应提供直观的导航、美观和响应式界面。用户体验应与（或超过）Ulanzi Studio生态系统的整体美学和功能标准保持一致。


适龄内容
商品应适合广泛的受众，并特别注意保护儿童免受不当内容的侵害。如果您的商品包含可能不适合特定受众的内容，您的商品应告知并指导用户。


性能和资源效率
您的产品应针对性能进行优化（如适用），最大限度地减少对 CPU/GPU、电池寿命、存储和数据使用等设备资源的影响。

审查和重新评估
Ulanzi Studio保留随时审查和重新评估提交内容的权利。如果发现商品违反了这些指南，我们可能会将其移除或需要修改。
