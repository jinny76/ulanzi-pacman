# Pac-Man Runner 吃豆人跑酷插件

一个有趣的Ulanzi Deck插件，让吃豆人在你的按键之间跑来跑去！

## ✨ 功能特性

- 🎮 **动态动画**: 吃豆人会在按键网格中自动移动
- 🍒 **游戏元素**: 显示豆子、吃豆人等经典游戏元素
- ⭐ **幸运豆系统**: 5%概率生成金色幸运豆，吃到后触发特殊效果
  - 🌈 **彩虹效果**: 动态彩虹渐变背景
  - ⚡ **闪烁效果**: 金黄色脉冲闪烁
  - 📏 **巨大效果**: 所有普通豆子变大
- 🎊 **马年祝福语**: 吃到幸运豆时，在4个专用格子上显示马年吉祥成语
  - 🐴 **马年主题**: 20个与马相关的吉祥成语（马到成功、龙马精神等）
  - ⏰ **定时显示**: 成语持续显示5秒后自动消失
  - 🎨 **金色渐变**: 华丽的金色渐变背景突出显示成语
  - 📊 **按序排列**: 4个格子按位置0-3依次显示成语的4个字
- 🎬 **稳定动画**: 8 FPS稳定帧率，约1秒移动1格，避免通信阻塞
- 🧠 **智能寻路**: 吃豆人会主动寻找并吃掉最近的豆子
- ⚙️ **灵活配置**: 支持不同的网格布局（2x5, 3x3, 3x5等）
- 🎯 **交互控制**: 点击任意按键可暂停/继续游戏
- ✅ **智能验证**: 自动检测坐标冲突，确保配置正确
- 🌐 **全局设置**: 支持全局配置网格大小，自动重新初始化

## 📦 安装方法

1. 将整个 `com.ulanzi.pacman.ulanziPlugin` 文件夹复制到 UlanziStudio 插件目录：
   - **Windows**: `%APPDATA%\UlanziStudio\plugins\`
   - **macOS**: `~/Library/Application Support/UlanziStudio/plugins/`

2. 重启 UlanziStudio

3. 在插件列表中找到 "Pac-Man Runner"

## 🎮 使用步骤

### 步骤0: 配置网格大小（全局设置）

在插件设置中配置网格大小（默认3x3）：

1. 打开 UlanziStudio 插件设置
2. 找到 "Pac-Man Runner" 插件
3. 设置行数（Rows）和列数（Columns）
   - **3行3列**：标准按键布局（推荐）⭐
   - **3行5列**：适合D200竖向布局
   - **2行5列**：适合D200横向布局

**注意**: 修改网格大小后，需要重新配置所有按键的位置。

### 步骤1: 布局按键

将 "Pac-Man Cell" Action 拖到多个按键上，建议使用以下布局：

**3x3 网格**（推荐）⭐:
```
┌─────┬─────┬─────┐
│ 0,0 │ 0,1 │ 0,2 │
├─────┼─────┼─────┤
│ 1,0 │ 1,1 │ 1,2 │
├─────┼─────┼─────┤
│ 2,0 │ 2,1 │ 2,2 │
└─────┴─────┴─────┘
```

**3x5 网格**（D200竖向）:
```
┌─────┬─────┬─────┬─────┬─────┐
│ 0,0 │ 0,1 │ 0,2 │ 0,3 │ 0,4 │
├─────┼─────┼─────┼─────┼─────┤
│ 1,0 │ 1,1 │ 1,2 │ 1,3 │ 1,4 │
├─────┼─────┼─────┼─────┼─────┤
│ 2,0 │ 2,1 │ 2,2 │ 2,3 │ 2,4 │
└─────┴─────┴─────┴─────┴─────┘
```

**2x5 网格**（D200横向）:
```
┌─────┬─────┐
│ 0,0 │ 0,1 │
├─────┼─────┤
│ 1,0 │ 1,1 │
├─────┼─────┤
│ 2,0 │ 2,1 │
├─────┼─────┤
│ 3,0 │ 3,1 │
├─────┼─────┤
│ 4,0 │ 4,1 │
└─────┴─────┘
```

### 步骤2: 配置位置

点击每个按键，打开配置界面：

1. **行 (Row)**: 输入该按键在网格中的行号（从0开始）
2. **列 (Column)**: 输入该按键在网格中的列号（从0开始）

**示例**：
- 左上角按键：行=0, 列=0
- 右上角按键（3x3）：行=0, 列=2
- 中心按键（3x3）：行=1, 列=1

**⚠️ 重要提示**：
- 至少需要配置 **2个按键** 才能启动游戏
- 每个按键的坐标必须 **唯一**，不能有重复
- 如果检测到坐标冲突，按键会显示 **红色错误图标** 并提示修改
- 未配置的按键会显示 **黄色提示图标**

### 步骤3: 配置祝福语格子（可选）

如果想要显示马年吉祥成语，需要配置4个"祝福语"格子：

1. 将 "祝福语" Action 拖到4个按键上（建议放在右侧）
2. 分别配置为位置0、1、2、3：
   - 第1个按键：位置0（显示成语第1个字）
   - 第2个按键：位置1（显示成语第2个字）
   - 第3个按键：位置2（显示成语第3个字）
   - 第4个按键：位置3（显示成语第4个字）

**示例布局**（3x5网格）:
```
┌─────┬─────┬─────┬─────┬─────┐
│吃豆人│吃豆人│吃豆人│祝福0│祝福1│
├─────┼─────┼─────┼─────┼─────┤
│吃豆人│吃豆人│吃豆人│祝福2│祝福3│
├─────┼─────┼─────┼─────┼─────┤
│吃豆人│吃豆人│吃豆人│     │     │
└─────┴─────┴─────┴─────┴─────┘
```

**触发条件**：
- 吃豆人吃到金色幸运豆时
- 4个祝福语格子同时显示一个马年成语（如"马到成功"）
- 成语显示5秒后自动消失

### 步骤4: 享受游戏

配置完成后（所有坐标都不同且至少2个按键），吃豆人会自动开始移动！

- **暂停/继续**: 点击任意吃豆人按键
- **重新开始**: 暂停后再次点击
- **成语显示**: 吃到金色幸运豆时自动触发

## 🎨 视觉效果

- **黄色吃豆人**: 会根据移动方向旋转，并有张嘴闭嘴的动画
- **粉色豆子**: 散布在各个格子中
- **金色幸运豆**: 5%概率生成，带呼吸闪烁效果和外发光
- **特殊效果**:
  - 🌈 彩虹效果：动态彩虹渐变背景，持续3秒
  - ⚡ 闪烁效果：金黄色脉冲闪烁，持续3秒
  - 📏 巨大效果：所有普通豆子放大1.5倍，持续3秒
- **智能刷新**: 吃完所有豆子后会停下2秒，然后重新生成豆子
- **黑色背景**: 带有细微的网格线
- **连贯画面**: 所有按键组合成一个完整的游戏场景

## ⚙️ 技术特点

- **游戏引擎**: 独立的游戏逻辑引擎，支持智能寻路和目标导向移动
- **渲染系统**: 在完整Canvas上绘制，然后切分给各个按键
- **性能优化**: 8 FPS稳定帧率 + 智能渲染（仅更新变化的格子）+ 避免通信阻塞
- **智能寻路**: 曼哈顿距离算法，吃豆人会主动寻找豆子
- **随机性增强**:
  - 目标选择：70%选最近的，30%随机选择前3个之一
  - 移动方向：80%朝向目标，20%完全随机（包括回头）
  - 随机游荡：吃到豆子后40%概率随机游荡1-2步
- **幸运豆系统**:
  - 5%概率生成金色幸运豆
  - 吃到后随机触发3种特殊效果之一（彩虹/闪烁/巨大）
  - 特殊效果持续3秒，带渐淡过渡
- **马年祝福语系统**:
  - 20个与马相关的吉祥成语库
  - 检测幸运豆事件，触发成语显示
  - 4个格子按位置0-3依次显示成语的4个字
  - 金色渐变背景 + 大字号显示
  - 5秒自动消失，恢复黑屏
- **豆子刷新**: 吃完所有豆子后停下2秒，然后自动刷新并继续游戏
- **多Action协同**: 展示了多个Action如何共享状态和协同工作
- **坐标验证**: 自动检测并提示坐标冲突，防止配置错误
- **视觉反馈**: 不同状态显示不同图标（未配置/冲突/正常）
- **动态配置**: 支持全局设置网格大小，自动重新初始化游戏引擎
