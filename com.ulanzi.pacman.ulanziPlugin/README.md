# Pac-Man Runner 吃豆人跑酷插件

一个有趣的Ulanzi Deck插件，让吃豆人在你的按键之间跑来跑去！

## ✨ 功能特性

- 🎮 **动态动画**: 吃豆人会在按键网格中自动移动
- 🍒 **游戏元素**: 显示豆子、吃豆人等经典游戏元素
- ⭐ **幸运豆系统**: 随机生成金色幸运豆，吃到后触发特殊效果
  - 🌈 **彩虹效果**: 动态彩虹渐变背景
  - ⚡ **闪烁效果**: 金黄色脉冲闪烁
  - 📏 **巨大效果**: 所有普通豆子变大
- 🎊 **马年祝福语**: 吃到幸运豆时，在专用格子上显示马年吉祥成语
  - 🐴 **马年主题**: 20个与马相关的吉祥成语（马到成功、龙马精神等）
  - ⏰ **定时显示**: 成语持续显示5秒后自动消失
  - 🎨 **金色渐变**: 华丽的金色渐变背景突出显示成语
  - 📊 **按序排列**: 4个字按顺序依次显示
- 🎬 **稳定动画**: 8 FPS稳定帧率，约1秒移动1格
- 🧠 **智能寻路**: 吃豆人会主动寻找并吃掉最近的豆子
- 🎯 **交互控制**: 点击任意游戏按键可暂停/继续游戏
- ⚡ **零配置**: 自动识别按键位置，拖拽即用

## 📦 安装方法

1. 将整个 `com.ulanzi.pacman.ulanziPlugin` 文件夹复制到 UlanziStudio 插件目录：
   - **Windows**: `%APPDATA%\UlanziStudio\plugins\`
   - **macOS**: `~/Library/Application Support/UlanziStudio/plugins/`

2. 重启 UlanziStudio

3. 在插件列表中找到 "Pac-Man Runner"

## 🎮 使用步骤

### 超简单！只需拖拽 15 个按键 🎉

将 "Pac-Man Runner" Action 拖到设备的 **3行5列** 按键网格上：

```
       列0   列1   列2   列3   列4
行0:   [🟡] [🟡] [🟡] [马] [到]
行1:   [🟡] [🟡] [🟡] [成] [功]
行2:   [🟡] [🟡] [🟡] [  ] [  ]
```

**布局说明**:
- **左侧 3x3 区域** (列0-2，行0-2)：游戏区域，吃豆人在此移动
- **右侧 4 个按键** (列3-4，行0-1)：幸运词显示区域，吃到幸运豆时显示成语

**就这么简单！** ✨
- ✅ 无需手动配置坐标
- ✅ 自动识别按键位置
- ✅ 拖拽完成后立即开始游戏

### 游戏控制

- **暂停/继续**: 点击任意游戏区域按键（左侧 3x3）
- **观察成语**: 吃到金色幸运豆时，右侧自动显示马年吉祥成语

## 🎨 视觉效果

- **黄色吃豆人**: 会根据移动方向旋转，并有张嘴闭嘴的动画
- **粉色豆子**: 散布在各个格子中
- **金色幸运豆**: 随机生成，带呼吸闪烁效果和外发光
- **特殊效果**:
  - 🌈 彩虹效果：动态彩虹渐变背景，持续3秒
  - ⚡ 闪烁效果：金黄色脉冲闪烁，持续3秒
  - 📏 巨大效果：所有普通豆子放大，持续3秒
- **智能刷新**: 吃完所有豆子后会停下2秒，然后重新生成豆子
- **黑色背景**: 带有细微的网格线
- **连贯画面**: 所有按键组合成一个完整的游戏场景

## 📐 按键布局详解

### 固定 3x5 布局

本插件使用 **固定的 3行5列** 布局，系统根据按键位置自动识别角色：

```
设备坐标系统（SDK key 格式："列_行"）

"0_0"  "1_0"  "2_0"  "3_0"  "4_0"  ← 行0
"0_1"  "1_1"  "2_1"  "3_1"  "4_1"  ← 行1
"0_2"  "1_2"  "2_2"  "3_2"  "4_2"  ← 行2
  ↑      ↑      ↑      ↑      ↑
 列0    列1    列2    列3    列4
```

**自动分区规则**:
- `列0-2，行0-2` → 游戏区域（9个格子，3x3游戏网格）
- `列3行0，列4行0，列3行1，列4行1` → 幸运词显示（4个字）
- 其他位置 → 未使用（可以不放置）

**示例**:
- 按键 "0_0" (列0行0) → 游戏区域左上角
- 按键 "2_2" (列2行2) → 游戏区域右下角
- 按键 "3_0" (列3行0) → 幸运词第1个字
- 按键 "4_1" (列4行1) → 幸运词第4个字

## ⚙️ 技术特点

- **零配置设计**: 利用 SDK 的 key 参数自动识别按键位置
- **游戏引擎**: 独立的游戏逻辑引擎，支持智能寻路和目标导向移动
- **渲染系统**: 在完整Canvas上绘制，然后切分给各个按键
- **性能优化**: 8 FPS稳定帧率 + 智能渲染（仅更新变化的格子）
- **智能寻路**: 曼哈顿距离算法，吃豆人会主动寻找豆子
- **随机性增强**:
  - 目标选择：70%选最近的，30%随机选择前3个之一
  - 移动方向：80%朝向目标，20%完全随机（包括回头）
  - 随机游荡：吃到豆子后40%概率随机游荡1-2步
- **幸运豆系统**:
  - 每次刷新确保至少生成1个金色幸运豆
  - 吃到后随机触发3种特殊效果之一（彩虹/闪烁/巨大）
  - 特殊效果持续3秒，带渐淡过渡
- **马年祝福语系统**:
  - 20个与马相关的吉祥成语库
  - 检测幸运豆事件，触发成语显示
  - 4个格子按顺序依次显示成语的4个字
  - 金色渐变背景 + 大字号显示
  - 5秒自动消失，恢复黑屏
- **豆子刷新**: 吃完所有豆子后停下2秒，然后自动刷新并继续游戏
